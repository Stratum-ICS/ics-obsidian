## Key Decisions Made

| Decision | Rationale | Alternatives Considered |
|----------|-----------|-------------------------|
| Canonical templates + skill in **`ics-agents`**, not only plugin | Rules change faster than plugin releases; single source of truth | Plugin-only `instruction.md` |
| **`instruction.md` next to `hub.md`**, skill materializes on bootstrap | Human contract visible in vault | Generated only inside plugin |
| Commit line: `[actor][research][paper_id][phase] summary` | Greppable `ics log`, multi-actor teams | Conventional Commits only |
| Log filter: buffer full `ics log` then **substring** lines | `ics` path args unknown in CI; works everywhere | CLI passthrough (deferred until `ics log --help` confirms) |
| Hub frontmatter: **`kind: hub`**, not `phase: hub` | Matches `instruction.md` phase table for non-hub notes | `phase: hub` |
| **`pdf_rel_path` always vault-root-relative** | Same string in all notes + wikilinks | Mixed relative roots |
